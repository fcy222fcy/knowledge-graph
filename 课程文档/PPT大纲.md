# PPT大纲

## 项目名称

SE智图问答平台（软件工程课程智能教学辅助系统）

## 项目简介

本项目是一款面向软件工程课程的智能教学辅助系统，旨在通过知识图谱与AI技术提升学习效率。系统支持课程文档自动解析、知识图谱可视化、基于RAG的智能问答、自动测验与学习分析，帮助学生更好地理解和掌握软件工程知识体系。

## 项目开发软件

- **前端**：Vue 3 + Vite + Element Plus + D3.js
- **后端**：Go + Gin + GORM
- **数据库**：MySQL+Neo4j
- **AI服务**：Ollama（本地大模型）
- **工具**：Git、VS Code、Postman

## 项目主要功能

1. **知识图谱管理**：上传课程文档，自动抽取知识点并可视化展示
2. **AI智能问答**：基于RAG技术，结合知识图谱和文档进行精准问答
3. **测验系统**：支持单选/多选题，自动批改并统计成绩
4. **学习统计**：分析知识点掌握度、学习趋势，识别薄弱环节
5. **文件管理**：上传和管理课程文档，作为知识图谱构建来源
