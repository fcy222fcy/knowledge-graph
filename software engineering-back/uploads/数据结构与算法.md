# 数据结构与算法详解

## 一、线性数据结构

### 1.1 数组（Array）

**特点**：连续存储，随机访问 O(1)，插入删除 O(n)。

**动态数组（Go slice）底层原理**：
```
┌─────────────────────────────────┐
│  指向底层数组的指针 | len | cap  │  ← slice header（24字节）
└─────────────────────────────────┘
           │
           ▼
┌───┬───┬───┬───┬───┬───┬───┬───┐
│ 0 │ 1 │ 2 │ 3 │ 4 │   │   │   │  ← 底层数组
└───┴───┴───┴───┴───┴───┴───┴───┘
  len=5  cap=8

扩容策略：
- 当 cap < 256 时，新 cap = 2 * 旧 cap
- 当 cap >= 256 时，新 cap = 旧 cap + 旧cap/4 + 196
```

### 1.2 链表（Linked List）

**单链表**：
```
[1|→] → [2|→] → [3|→] → [4|∅]
 data    data    data    data
```

**双向链表**：
```
[∅|1|→] ⇄ [←|2|→] ⇄ [←|3|→] ⇄ [←|4|∅]
```

**循环链表**：
```
[1|→] → [2|→] → [3|→] → [4|→] → 回到[1]
```

**时间复杂度对比**：

| 操作 | 数组 | 单链表 | 双向链表 |
|------|------|--------|---------|
| 按索引访问 | O(1) | O(n) | O(n) |
| 头部插入 | O(n) | O(1) | O(1) |
| 尾部插入 | O(1)* | O(n) | O(1)** |
| 按值查找 | O(n) | O(n) | O(n) |
| 删除 | O(n) | O(n) | O(1)*** |

*均摊；**有尾指针时；***有节点引用时

### 1.3 栈（Stack）

**LIFO（后进先出）**：

```
操作：push(1), push(2), push(3), pop(), pop()

    │ 3 │ ← push(3) 后
    │ 2 │ ← push(2) 后
    │ 1 │ ← push(1) 后
    └───┘

pop() → 3（后进先出）
pop() → 2
```

**应用场景**：
- 函数调用栈
- 括号匹配
- 表达式求值
- 浏览器前进/后退
- 深度优先搜索（DFS）

### 1.4 队列（Queue）

**FIFO（先进先出）**：

```
入队 → [1] [2] [3] [4] → 出队
        front         rear
```

**变种**：

| 类型 | 特点 | 应用 |
|------|------|------|
| 双端队列（Deque） | 两端都可进出 | 滑动窗口 |
| 优先队列 | 元素按优先级出队 | 任务调度、Dijkstra |
| 循环队列 | 数组实现，头尾循环 | 操作系统缓冲区 |

**Go 实现循环队列**：
```go
type CircularQueue struct {
    data  []interface{}
    head  int
    tail  int
    size  int
    cap   int
}

func NewCircularQueue(cap int) *CircularQueue {
    return &CircularQueue{
        data: make([]interface{}, cap),
        cap:  cap,
    }
}

func (q *CircularQueue) Enqueue(val interface{}) bool {
    if q.size == q.cap {
        return false // 队列满
    }
    q.data[q.tail] = val
    q.tail = (q.tail + 1) % q.cap
    q.size++
    return true
}

func (q *CircularQueue) Dequeue() (interface{}, bool) {
    if q.size == 0 {
        return nil, false // 队列空
    }
    val := q.data[q.head]
    q.head = (q.head + 1) % q.cap
    q.size--
    return val, true
}
```

## 二、树形数据结构

### 2.1 二叉树（Binary Tree）

```
        1          ← 根节点
       / \
      2   3        ← 第二层
     / \ / \
    4  5 6  7      ← 第三层（叶子节点）
```

**性质**：
- 第 i 层最多有 2^(i-1) 个节点
- 深度为 k 的树最多有 2^k - 1 个节点
- 叶子节点数 n0 = 度为2的节点数 n2 + 1（即 n0 = n2 + 1）

**遍历方式**：
```go
type TreeNode struct {
    Val   int
    Left  *TreeNode
    Right *TreeNode
}

// 前序遍历：根 → 左 → 右
func preorder(root *TreeNode) []int {
    if root == nil {
        return nil
    }
    result := []int{root.Val}
    result = append(result, preorder(root.Left)...)
    result = append(result, preorder(root.Right)...)
    return result
}

// 中序遍历：左 → 根 → 右（BST 中是有序的）
func inorder(root *TreeNode) []int {
    if root == nil {
        return nil
    }
    result := inorder(root.Left)
    result = append(result, root.Val)
    result = append(result, inorder(root.Right)...)
    return result
}

// 后序遍历：左 → 右 → 根（适用于释放内存）
func postorder(root *TreeNode) []int {
    if root == nil {
        return nil
    }
    result := postorder(root.Left)
    result = append(result, postorder(root.Right)...)
    result = append(result, root.Val)
    return result
}

// 层序遍历（BFS）
func levelOrder(root *TreeNode) [][]int {
    if root == nil {
        return nil
    }
    var result [][]int
    queue := []*TreeNode{root}
    for len(queue) > 0 {
        levelSize := len(queue)
        var level []int
        for i := 0; i < levelSize; i++ {
            node := queue[0]
            queue = queue[1:]
            level = append(level, node.Val)
            if node.Left != nil {
                queue = append(queue, node.Left)
            }
            if node.Right != nil {
                queue = append(queue, node.Right)
            }
        }
        result = append(result, level)
    }
    return result
}
```

### 2.2 二叉搜索树（BST）

**性质**：左子树所有节点 < 根节点 < 右子树所有节点

```
        8
       / \
      3   10
     / \    \
    1   6    14
       / \   /
      4   7 13
```

**操作时间复杂度**：

| 操作 | 平均 | 最坏（退化为链表） |
|------|------|-------------------|
| 查找 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

**BST 转有序数组**：中序遍历即可。

### 2.3 平衡二叉树（AVL）

AVL 树通过旋转操作保持任意节点的左右子树高度差不超过1。

**四种旋转**：

```
LL型（右旋）：       RR型（左旋）：      LR型（先左旋再右旋）：  RL型（先右旋再左旋）：
    30                  10                  30                    10
   /                   / \                    \                  / \
  20                 20   30                  20                10  30
 /                  /    /                   /                      /
10                 10   40                  10                    20
```

### 2.4 红黑树

红黑树是一种自平衡 BST，通过颜色标记和旋转保持近似平衡。

**红黑树性质**：
1. 每个节点是红色或黑色
2. 根节点是黑色
3. 所有叶子节点（NIL）是黑色
4. 红色节点的子节点必须是黑色
5. 从任一节点到其所有叶子的路径上，黑色节点数相同

**应用**：
- Java TreeMap / TreeMap
- Linux CFS 进程调度
- epoll 中的 fd 管理

### 2.5 B 树与 B+ 树

**B 树**（多路平衡搜索树）：
```
           [10 | 20]
          /    |    \
    [3|5|7] [12|15] [22|25|30]
```

**B+ 树**（数据库索引常用）：
```
          [10 | 20]
         /    |    \
  [3|5|7] → [12|15] → [22|25|30]  ← 叶子节点链表连接
     ↓          ↓           ↓
   数据指针    数据指针    数据指针
```

**B+ 树优势**：
1. 非叶子节点只存键，扇出更大，树更矮
2. 叶子节点链表连接，范围查询高效
3. 所有查询都到叶子节点，查询性能稳定

## 三、哈希表（Hash Table）

### 3.1 基本原理

```
Key → Hash函数 → 数组索引 → Value

例: hash("name") % 16 = 5
    table[5] = "Alice"
```

### 3.2 冲突解决

**链地址法（拉链法）**：
```
[0] → NULL
[1] → [key1|val1] → [key2|val2] → NULL
[2] → [key3|val3] → NULL
[3] → NULL
```

**开放定址法（线性探测）**：
```
hash("a") = 3 → table[3] = a
hash("b") = 3 → table[3]已满 → table[4] = b
hash("c") = 3 → table[3,4]已满 → table[5] = c
```

### 3.3 负载因子与扩容

```
负载因子 α = 已存元素数 / 表长

α < 0.75: 性能良好
α > 0.75: 冲突概率增大，考虑扩容

扩容: 创建 2倍大小的新表，重新哈希所有元素
```

### 3.4 常见哈希函数

| 函数 | 公式 | 特点 |
|------|------|------|
| 除留余数法 | h(key) = key % p | 简单，p取质数 |
| 乘法散列法 | h(key) = ⌊m(kA mod 1)⌋ | 0 < A < 1 |
| SHA-256 | 复杂算法 | 密码学安全 |
| MurmurHash | 非加密哈希 | 高性能，分布均匀 |

## 四、图算法

### 4.1 图的表示

**邻接矩阵**（适合稠密图）：
```
    0  1  2  3
0 [ 0, 1, 0, 1 ]
1 [ 1, 0, 1, 0 ]
2 [ 0, 1, 0, 1 ]
3 [ 1, 0, 1, 0 ]

空间: O(V²)
判断边是否存在: O(1)
遍历邻居: O(V)
```

**邻接表**（适合稀疏图）：
```
0 → [1] → [3]
1 → [0] → [2]
2 → [1] → [3]
3 → [0] → [2]

空间: O(V + E)
判断边是否存在: O(度数)
遍历邻居: O(度数)
```

### 4.2 最短路径算法

#### Dijkstra 算法（非负权图）

```go
func dijkstra(graph [][]Edge, start int) []int {
    n := len(graph)
    dist := make([]int, n)
    for i := range dist {
        dist[i] = math.MaxInt32
    }
    dist[start] = 0

    // 优先队列（最小堆）
    pq := &PriorityQueue{}
    heap.Push(pq, Item{node: start, dist: 0})

    for pq.Len() > 0 {
        item := heap.Pop(pq).(Item)
        u := item.node

        if item.dist > dist[u] {
            continue // 已找到更短路径
        }

        for _, edge := range graph[u] {
            v := edge.To
            if dist[u]+edge.Weight < dist[v] {
                dist[v] = dist[u] + edge.Weight
                heap.Push(pq, Item{node: v, dist: dist[v]})
            }
        }
    }
    return dist
}
```

时间复杂度：O((V + E) log V)（使用优先队列）

#### Bellman-Ford 算法（含负权边）

```go
func bellmanFord(n int, edges []Edge, start int) ([]int, bool) {
    dist := make([]int, n)
    for i := range dist {
        dist[i] = math.MaxInt32
    }
    dist[start] = 0

    // 松弛 V-1 次
    for i := 0; i < n-1; i++ {
        for _, edge := range edges {
            if dist[edge.From] != math.MaxInt32 &&
                dist[edge.From]+edge.Weight < dist[edge.To] {
                dist[edge.To] = dist[edge.From] + edge.Weight
            }
        }
    }

    // 检测负权环
    for _, edge := range edges {
        if dist[edge.From] != math.MaxInt32 &&
            dist[edge.From]+edge.Weight < dist[edge.To] {
            return nil, false // 存在负权环
        }
    }
    return dist, true
}
```

时间复杂度：O(V × E)

#### Floyd-Warshall（全源最短路）

```go
func floyd(n int, dist [][]int) [][]int {
    for k := 0; k < n; k++ {
        for i := 0; i < n; i++ {
            for j := 0; j < n; j++ {
                if dist[i][k]+dist[k][j] < dist[i][j] {
                    dist[i][j] = dist[i][k] + dist[k][j]
                }
            }
        }
    }
    return dist
}
```

时间复杂度：O(V³)

### 4.3 最小生成树

#### Kruskal 算法（贪心 + 并查集）

```go
func kruskal(n int, edges []Edge) []Edge {
    // 按权值排序
    sort.Slice(edges, func(i, j int) bool {
        return edges[i].Weight < edges[j].Weight
    })

    uf := NewUnionFind(n)
    var mst []Edge

    for _, edge := range edges {
        if uf.Find(edge.From) != uf.Find(edge.To) {
            uf.Union(edge.From, edge.To)
            mst = append(mst, edge)
            if len(mst) == n-1 {
                break
            }
        }
    }
    return mst
}
```

#### Prim 算法（贪心扩展）

```go
func prim(graph [][]Edge, n int) []Edge {
    inMST := make([]bool, n)
    minEdge := make([]int, n)    // 到MST的最小边权
    parent := make([]int, n)     // 父节点
    for i := range minEdge {
        minEdge[i] = math.MaxInt32
        parent[i] = -1
    }
    minEdge[0] = 0

    var mst []Edge
    for i := 0; i < n; i++ {
        // 找不在MST中的最小权值节点
        u := -1
        for v := 0; v < n; v++ {
            if !inMST[v] && (u == -1 || minEdge[v] < minEdge[u]) {
                u = v
            }
        }
        inMST[u] = true
        if parent[u] != -1 {
            mst = append(mst, Edge{From: parent[u], To: u, Weight: minEdge[u]})
        }
        // 更新邻居
        for _, edge := range graph[u] {
            if !inMST[edge.To] && edge.Weight < minEdge[edge.To] {
                minEdge[edge.To] = edge.Weight
                parent[edge.To] = u
            }
        }
    }
    return mst
}
```

### 4.4 拓扑排序

用于有向无环图（DAG）的线性排序：

```go
func topologicalSort(n int, graph [][]int) []int {
    inDegree := make([]int, n)
    for i := 0; i < n; i++ {
        for _, v := range graph[i] {
            inDegree[v]++
        }
    }

    var queue []int
    for i := 0; i < n; i++ {
        if inDegree[i] == 0 {
            queue = append(queue, i)
        }
    }

    var result []int
    for len(queue) > 0 {
        u := queue[0]
        queue = queue[1:]
        result = append(result, u)
        for _, v := range graph[u] {
            inDegree[v]--
            if inDegree[v] == 0 {
                queue = append(queue, v)
            }
        }
    }

    if len(result) != n {
        return nil // 存在环
    }
    return result
}
```

## 五、动态规划（Dynamic Programming）

### 5.1 核心思想

**适用条件**：
1. **最优子结构**：问题的最优解包含子问题的最优解
2. **重叠子问题**：子问题被重复计算

**解题步骤**：
1. 定义状态（dp[i][j] 代表什么）
2. 推导状态转移方程
3. 确定初始值和边界
4. 确定遍历顺序
5. （可选）空间优化

### 5.2 经典问题

#### 0-1 背包问题

```
有 n 个物品，每个物品有重量 w[i] 和价值 v[i]
背包容量为 W，求最大价值

状态: dp[i][j] = 前i个物品，容量为j时的最大价值
转移: dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i])
初始: dp[0][j] = 0

空间优化（滚动数组）:
dp[j] = max(dp[j], dp[j-w[i]] + v[i])  // 逆序遍历j
```

```go
func knapsack(W int, weights, values []int) int {
    n := len(weights)
    dp := make([]int, W+1)

    for i := 0; i < n; i++ {
        for j := W; j >= weights[i]; j-- {
            dp[j] = max(dp[j], dp[j-weights[i]]+values[i])
        }
    }
    return dp[W]
}
```

#### 最长公共子序列（LCS）

```
s1 = "ABCBDAB"
s2 = "BDCAB"
LCS = "BCAB"（长度4）

dp[i][j] = s1前i个字符和s2前j个字符的LCS长度

转移:
  if s1[i] == s2[j]:
      dp[i][j] = dp[i-1][j-1] + 1
  else:
      dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

```go
func lcs(s1, s2 string) int {
    m, n := len(s1), len(s2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }

    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if s1[i-1] == s2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}
```

#### 最长递增子序列（LIS）

```go
func lis(nums []int) int {
    n := len(nums)
    dp := make([]int, n) // dp[i] = 以nums[i]结尾的LIS长度
    for i := range dp {
        dp[i] = 1
    }

    for i := 1; i < n; i++ {
        for j := 0; j < i; j++ {
            if nums[j] < nums[i] {
                dp[i] = max(dp[i], dp[j]+1)
            }
        }
    }

    result := 0
    for _, v := range dp {
        result = max(result, v)
    }
    return result
}

// O(n log n) 优化
func lisOptimized(nums []int) int {
    tails := []int{} // tails[i] = 长度为i+1的递增子序列的最小末尾元素
    for _, num := range nums {
        idx := sort.SearchInts(tails, num)
        if idx == len(tails) {
            tails = append(tails, num)
        } else {
            tails[idx] = num
        }
    }
    return len(tails)
}
```

#### 编辑距离

```
将 word1 转换为 word2 的最少操作数（插入、删除、替换）

dp[i][j] = word1前i个字符转换为word2前j个字符的最少操作数

转移:
  if word1[i] == word2[j]:
      dp[i][j] = dp[i-1][j-1]
  else:
      dp[i][j] = 1 + min(
          dp[i-1][j],    // 删除
          dp[i][j-1],    // 插入
          dp[i-1][j-1]   // 替换
      )
```

## 七、排序算法详解

### 7.1 快速排序

```go
func quickSort(arr []int, low, high int) {
    if low < high {
        pivot := partition(arr, low, high)
        quickSort(arr, low, pivot-1)
        quickSort(arr, pivot+1, high)
    }
}

func partition(arr []int, low, high int) int {
    pivot := arr[high] // 选择最后一个元素作为基准
    i := low - 1

    for j := low; j < high; j++ {
        if arr[j] < pivot {
            i++
            arr[i], arr[j] = arr[j], arr[i]
        }
    }
    arr[i+1], arr[high] = arr[high], arr[i+1]
    return i + 1
}
```

### 7.2 归并排序

```go
func mergeSort(arr []int) []int {
    if len(arr) <= 1 {
        return arr
    }
    mid := len(arr) / 2
    left := mergeSort(arr[:mid])
    right := mergeSort(arr[mid:])
    return merge(left, right)
}

func merge(left, right []int) []int {
    result := make([]int, 0, len(left)+len(right))
    i, j := 0, 0
    for i < len(left) && j < len(right) {
        if left[i] <= right[j] {
            result = append(result, left[i])
            i++
        } else {
            result = append(result, right[j])
            j++
        }
    }
    result = append(result, left[i:]...)
    result = append(result, right[j:]...)
    return result
}
```

### 7.3 堆排序

```go
func heapSort(arr []int) {
    n := len(arr)

    // 构建最大堆
    for i := n/2 - 1; i >= 0; i-- {
        heapify(arr, n, i)
    }

    // 逐个提取元素
    for i := n - 1; i > 0; i-- {
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)
    }
}

func heapify(arr []int, n, i int) {
    largest := i
    left := 2*i + 1
    right := 2*i + 2

    if left < n && arr[left] > arr[largest] {
        largest = left
    }
    if right < n && arr[right] > arr[largest] {
        largest = right
    }
    if largest != i {
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)
    }
}
```
