# SE智图问答 - MVP版本需求文档

## 一、项目概述

### 1.1 项目定位

SE智图问答是一个基于知识图谱的软件工程课程问答平台，帮助学生通过自然语言提问获取结构化知识，并提供知识图谱可视化辅助学习。

### 1.2 MVP目标

MVP（最小可行产品）聚焦于核心价值：**知识图谱 + 智能问答**，其他功能延后实现。

### 1.3 目标用户

- 软件工程专业学生
- 软件工程课程教师

## 四、数据模型设计

### 4.1 用户表 (users)

```sql
CREATE TABLE users (
    id          BIGINT PRIMARY KEY AUTO_INCREMENT,
    username    VARCHAR(50) UNIQUE NOT NULL COMMENT '用户名',
    password    VARCHAR(255) NOT NULL COMMENT '密码(加密)',
    email       VARCHAR(100) COMMENT '邮箱',
    avatar      VARCHAR(255) COMMENT '头像URL',
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at  DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 4.2 文件表 (files)

```sql
CREATE TABLE files (
    id          BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id     BIGINT NOT NULL COMMENT '上传用户',
    filename    VARCHAR(255) NOT NULL COMMENT '文件名',
    filepath    VARCHAR(500) NOT NULL COMMENT '存储路径',
    filetype    VARCHAR(20) NOT NULL COMMENT '文件类型(pdf/pptx/docx/md)',
    filesize    BIGINT COMMENT '文件大小(字节)',
    status      TINYINT DEFAULT 0 COMMENT '状态(0待解析/1解析中/2完成/3失败)',
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### 4.3 知识点表 (knowledge_nodes)

```sql
CREATE TABLE knowledge_nodes (
    id          BIGINT PRIMARY KEY AUTO_INCREMENT,
    name        VARCHAR(100) NOT NULL COMMENT '知识点名称',
    type        VARCHAR(50) COMMENT '类型(概念/方法/模型等)',
    description TEXT COMMENT '描述',
    file_id     BIGINT COMMENT '来源文件',
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (file_id) REFERENCES files(id)
);
```

### 4.4 知识关系表 (knowledge_edges)

```sql
CREATE TABLE knowledge_edges (
    id          BIGINT PRIMARY KEY AUTO_INCREMENT,
    source_id   BIGINT NOT NULL COMMENT '起始节点',
    target_id   BIGINT NOT NULL COMMENT '目标节点',
    relation    VARCHAR(50) NOT NULL COMMENT '关系类型(包含/依赖/前置等)',
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (source_id) REFERENCES knowledge_nodes(id),
    FOREIGN KEY (target_id) REFERENCES knowledge_nodes(id)
);
```

### 4.5 问答记录表 (qa_records)

```sql
CREATE TABLE qa_records (
    id          BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id     BIGINT NOT NULL COMMENT '用户',
    session_id  VARCHAR(50) COMMENT '会话ID',
    question    TEXT NOT NULL COMMENT '问题',
    answer      TEXT COMMENT '答案',
    source      JSON COMMENT '知识来源',
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### 4.6 学习统计表 (learning_stats)

```sql
CREATE TABLE learning_stats (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id         BIGINT NOT NULL COMMENT '用户',
    stat_date       DATE NOT NULL COMMENT '统计日期',
    study_duration  INT DEFAULT 0 COMMENT '学习时长(分钟)',
    qa_count        INT DEFAULT 0 COMMENT '问答次数',
    correct_count   INT DEFAULT 0 COMMENT '正确次数',
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    UNIQUE KEY uk_user_date (user_id, stat_date)
);
```

## 六、技术实现方案

### 6.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | HTML + CSS + JavaScript（或 Vue 3） |
| 后端 | Go + Gin + GORM |
| 数据库 | MySQL + Neo4j + Redis |
| 文件存储 | 本地文件系统（或 MinIO） |
| 图谱可视化 | vis.js / d3.js / cytoscape.js |
| LLM | OpenAI API / 其他大模型API |

## 七、验收标准

### 7.1 功能验收

- [ ] 用户可以注册、登录、退出
- [ ] 用户可以上传PDF/PPT/Word文件
- [ ] 用户可以看到知识图谱可视化
