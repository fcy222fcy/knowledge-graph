# 基于知识图谱的软件工程课程问答平台 — 基本文档

## 一、项目简介

本项目是一个面向软件工程课程的智能问答平台。系统支持上传课程相关资料（教材、课件、笔记、题库等），自动抽取知识点及其关系，构建软件工程课程知识图谱。学生可通过自然语言提问，系统基于知识图谱进行智能检索和回答，同时提供可视化学习功能。

核心目标：

1. 构建课程专属知识图谱
2. 提供结构化知识查询和问答
3. 支持知识可视化，辅助学习
4. 提高课程学习效率和理解能力

---

## 二、功能模块

### 1. 课程资料管理

**功能描述**：

* 上传各种课程资料：PDF、PPT、Word、Markdown、代码示例、题库等
* 文件存储及管理
* 文件解析和元信息记录

**技术实现**：

* Go + Gin：提供文件上传 API
* MySQL + GORM：存储文件元信息、课程信息
* 本地文件系统或对象存储（MinIO / S3）
* 文本解析库：

  * PDF：`unidoc`
  * PPT：`go-pptx`
  * DOCX/Markdown：标准 Go 文件解析
* Redis（可选）：缓存文件处理状态、进度

---

### 2. 文本解析与知识抽取

**功能描述**：

* 文本切分、清洗
* 分词处理
* 实体识别（知识点、概念、方法）
* 关系抽取（前置、包含、依赖等）

**技术实现**：

* Python NLP 服务（spaCy / LTP / Transformers）
* Go 调用 Python 服务（HTTP RPC）
* 异步处理任务队列（Redis Stream/List）
* 返回 JSON：实体 + 关系

> Go 可处理简单解析，但高级 NLP 建议使用 Python 微服务。

---

### 3. 知识图谱构建与存储

**功能描述**：

* 将实体和关系构建成图
* 存储节点和边
* 提供图查询接口

**技术实现**：

* 图数据库：Neo4j（推荐）
* Go 驱动：`neo4j-go-driver`
* MySQL + GORM（可选）：存储基础节点、元信息
* Redis：缓存热点节点、子图

---

### 4. 智能问答模块

**功能描述**：

* 用户自然语言提问
* 图谱检索相关知识点
* LLM 生成回答
* 返回答案及引用来源

**技术实现**：

* Go + Gin 提供问答接口
* 图谱查询：

  * Neo4j 查询子图
  * 可结合向量数据库（Milvus / Qdrant）做语义检索
* LLM 调用：

  * API 或自部署 LLM
* Redis：缓存热门问题答案

---

### 5. 可视化模块

**功能描述**：

* 知识图谱可视化
* 支持节点点击、展开、缩放

**技术实现**：

* Gin API 输出 JSON（节点列表 + 边列表）
* 前端可用 `vis.js` / `d3.js` / `cytoscape.js`
* Redis 缓存热点图数据

---

### 6. 用户与权限管理

**功能描述**：

* 用户注册、登录
* 角色权限管理
* Session 管理

**技术实现**：

* Gin + JWT
* MySQL + GORM
* Redis 缓存 token 和会话信息

---

## 三、整体数据流程

```
用户上传文件 (Gin + MySQL/MinIO)
      ↓
异步任务入队 (Redis Stream)
      ↓
Python NLP 服务处理文档 → 抽取实体 + 关系
      ↓
Go 接收 JSON → 存入 Neo4j / MySQL
      ↓
用户提问 (Gin API)
      ↓
查询图谱 + 向量检索 (Go + Neo4j / Milvus)
      ↓
调用 LLM 生成答案
      ↓
缓存热门答案 (Redis)
      ↓
返回前端
```

## 四、技术栈

| 模块     | 技术                                  |
| ------ | ----------------------------------- |
| 后端服务   | Go, Gin, GORM                       |
| 数据库    | MySQL（基础数据）、Neo4j（图谱）、Redis（缓存/队列）  |
| NLP 服务 | Python + spaCy / LTP / Transformers |
| 文件解析   | Go 库：unidoc, go-pptx, docx/markdown |
| 问答生成   | LLM API 或自部署大模型                     |
| 前端可视化  | vis.js / d3.js / cytoscape.js       |

---

## 五、项目亮点

1. 课程资料自动解析与知识图谱构建
2. 支持图谱查询 + LLM 问答结合
3. 高可扩展性：Go 高并发 + Python NLP 微服务
4. 可视化辅助学习，结构化呈现知识点
5. 可缓存热点问题，加速问答响应
