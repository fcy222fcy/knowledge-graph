说明： ① 本项目无独立Python FastAPI服务，AI能力通过Go直接调用本地Ollama API实现； ② 向量存储为自研内存实现（非FAISS），支持JSON持久化； ③ 图数据库采用Neo4j为主、MySQL为辅的双写策略，读取时Neo4j优先并自动回退MySQL； ④ 问答引擎实现5级降级策略：知识图谱查询 → 向量RAG → 关键词搜索 → 知识点匹配 → LLM自由回答； ⑤ 前端分为学生端和教师/管理员端两个独立Vue应用，共享后端API。
